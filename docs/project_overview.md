# OSINT-Based Security Assessment of a Simulated FinTech Organization

## Overview
This project demonstrates how Open-Source Intelligence (OSINT) techniques can be used to gather publicly available information about an organization and analyze potential security risks.

The investigation focuses on a simulated financial technology company named AstraFin Digital Payments Pvt Ltd.

The goal of this project is to demonstrate how attackers or security analysts can map an organization's digital footprint using publicly accessible sources.

---

## Objectives

• Demonstrate OSINT investigation techniques  
• Identify publicly exposed organizational data  
• Visualize relationships between digital assets  
• Assess potential security risks  
• Propose security recommendations
